# go-server-generator 

GoLang web server generator in nodejs

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```
npm install go-server-generator --save
```

## Run

```npm install```

```npm start```

## Tests

```npm install```

```npm test```
